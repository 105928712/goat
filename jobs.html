SDEV1 - Software Developer

This position requires the designing, building, testing, and maintenance of computer applications and programs. The role also involves translating user and client needs and requirements into functional code, while also identifying and resolving issues. Software Developers also ensure software performance, security, and reliability.

Salary Range: Up to $90,000 for entry level positions

Applicants Report To: Senior Software Developer (Level 3 Position)

Key Responsibilities:

  • To develop efficient and effective software that meets user and clients' needs
  • Ensuring that projects are secure with the necessary protections
  • Maintaining and updating existing software applications
  • Providing technical support to users and stakeholders
  • Testing software applications to identify and resolve bugs
  • Ensures software meets quality standards and performance requirements

Required Qualifications, Skills, Knowledge, and Attributes:

  • Experience with at least TWO of the following languages: JavaScript, HTML/CSS, Python, SQL, Java, C#
  • Knowledge of software development methodologies (Agile)
  • Familiarity with databases and data structures
  • Problem-solving and teamwork skills
  • Effective communication skills

NETA1 - Network Administrator

This position involves installing, maintaining, and troubleshooting computer networks and systems, ensuring smooth operation and security. This role also involves monitoring changes and providing troubleshooting support.

Salary Range: Up to $92,000 for entry level positions

Applicants Report To: Senior Network Administrator (Level 3 Position)

Key Responsibilities:

  • Installation and maintenance of network hardware and software
  • Maintain documentation of network configurations and troubleshooting steps
  • Implement and maintain security measures for network protection
  • Manage network hardware and software assets
  • Monitor network performance and resolve issues

Required Qualifications, Skills, Knowledge, and Attributes:

  • Proficiency with network hardware (routers, switches, firewalls, servers)
  • Experience with network monitoring tools, particularly SolarWinds
  • Problem-solving and teamwork skills
  • Effective communication skills

DAAN1 - Data Analyst

This position involves collecting, analyzing, and interpreting data to identify trends, solve problems, and inform company decisions. The role helps turn raw data into useful tools for forecasting.

Salary Range: Up to $85,000 for entry level positions

Applicants Report To: Senior Data Analyst (Level 3 Position)

Key Responsibilities:

  • Create and maintain information architecture for data sets
  • Make recommendations on how the company can best use data
  • Analyze data usage by other businesses in the industry
  • Integrate and analyze previously unrelated data sets

Required Qualifications, Skills, Knowledge, and Attributes:

  • Familiarity with databases and data structures
  • Problem-solving and teamwork skills
  • Effective communication skills
